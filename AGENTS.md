# AGENTS.md

## Objectif du projet
Ce dépôt contient l'application de gestion d'entreprise qui est une application de nextcloud de Benjamin AIMARD. L'application permet de gérer une base de données clients, faire des devis et des factures. De gérer une liste de produit et de faire une synthèse de ce qui a été facturé.

## Règles obligatoires
- Ne jamais pousser directement sur main.
- Créer une branche codex/<sujet>.
- Ouvrir une Pull Request claire.
- Ne pas modifier les données utilisateur.
- Ne pas supprimer de fonctionnalités existantes sans justification.
- Ajouter ou mettre à jour les tests si le comportement change.
- Conserver une documentation des commandes principales.
- Avant toute PR, vérifier que le programme démarre sans erreur.

## Priorités
1. Sécurité
2. Stabilité
3. Simplicité d’usage
